# Takeoff Media Interview
## Getting Started

1. Clone the repo
   ```sh
   git clone https://github.com/MauroPalgi/takeoff-media-interview.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
 3. Fix audited pacakges 
   ```sh
   npm audit fif
   ```
4. Start Gulp Server
   ```sh
   npm start
   ```
